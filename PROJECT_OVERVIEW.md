PROJECT OVERVIEW

Title:
Marketing ROI & Budget Reallocation Analysis

Description:
This project analyzes marketing campaign performance using a Python ETL pipeline and ROI analytics to evaluate marketing effectiveness and optimize advertising budget allocation.

Key Components:
• Python ETL pipeline to process raw marketing data
• Data transformation and aggregation using Pandas
• Campaign revenue and ROI analysis
• Visualization using Matplotlib
• Excel dashboard for marketing performance insights
• PowerPoint presentation and project report

Key Insights:
• Campaign C039 generated the highest ROI (~15x)
• Several campaigns delivered strong ROI between 4x–6x
• Some campaigns showed ROI below 2x indicating inefficient marketing spend
• Budget allocation should prioritize high-performing campaigns

Technology Stack:
Python, Pandas, Matplotlib, Excel Dashboard, VS Code
