HOW TO RUN THE PROJECT

1. Ensure Python is installed on the system.

2. Install required libraries:
   pip install pandas matplotlib

3. Navigate to the ETL folder.

4. Run the ETL pipeline:
   python etl_pipeline.py

5. Run the analysis script:
   python roi_analysis.py

6. Run visualization script:
   python visualize_roi.py

7. The generated outputs will be available in the analysis folder:
   - fact_sessions.csv
   - fact_campaign_daily.csv
   - fact_channel_daily.csv
   - campaign_roi_summary.csv
   - campaign_roi_chart.png

8. Open the Excel dashboard:
   marketing_roi_dashboard.xlsx

9. Review the PowerPoint presentation:
   Marketing_ROI_Presentation.pptx
