# Marketing ROI & Budget Reallocation Analysis

## Project Overview

This capstone project analyzes marketing campaign performance and recommends budget reallocation strategies to maximize Return on Investment (ROI).

## Business Problem

Marketing budgets are often allocated without complete visibility into campaign effectiveness. This project identifies high-performing campaigns and optimization opportunities.

## Objectives

- Analyze campaign performance
- Measure ROI across channels
- Identify underperforming campaigns
- Recommend budget reallocation strategies

## Dataset Used

- sessions.csv
- orders.csv
- order_items.csv
- campaigns.csv
- ad_spend_daily.csv
