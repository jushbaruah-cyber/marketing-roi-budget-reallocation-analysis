# Marketing ROI & Budget Reallocation Analysis

A data engineering and marketing analytics project that builds a Python ETL pipeline to evaluate campaign performance and identify opportunities for marketing budget optimization.

---

# Project Objective

The goal of this project is to analyze marketing campaign performance and measure Return on Investment (ROI) across different campaigns and marketing channels.

The project builds a complete data pipeline that transforms raw marketing data into analytical datasets and generates insights to guide budget allocation decisions.

---

# North Star Metric

The primary North Star metric used in this project is **Marketing Return on Investment (ROI)**.

ROI measures the effectiveness of marketing spend by comparing the revenue generated by campaigns relative to the advertising budget invested.

**ROI Formula**

ROI = Revenue / Ad Spend

Key KPI values from the analysis:

Total Revenue: **110.12M**
Total Ad Spend: **74.46M**
Average Campaign ROI: **4.52**
Best Performing Campaign: **C039**

This metric serves as the primary indicator for evaluating campaign efficiency and guiding marketing budget reallocation decisions.

# Dataset Description

| Dataset            | Description                                                   |
| ------------------ | ------------------------------------------------------------- |
| sessions.csv       | Website sessions including user, device, channel and campaign |
| orders.csv         | Purchase orders generated from sessions                       |
| order_items.csv    | Product details and price information                         |
| campaigns.csv      | Campaign metadata                                             |
| ad_spend_daily.csv | Daily advertising spend per campaign                          |

# Data Pipeline Architecture

The pipeline follows a simple analytics architecture:

Raw Data (CSV)
↓
ETL Pipeline (Python)
↓
Fact Tables
↓
ROI Analysis
↓
Visualization & Insights

---

# ETL Process

The ETL pipeline performs the following steps:

## Extract

Load raw CSV files using **Pandas**.

## Transform

Key transformations include:

* Join sessions, orders, and order_items datasets
* Compute revenue per order
* Create fact tables for campaign and channel analysis
* Aggregate revenue and orders per day
* Prepare clean analytical datasets for ROI analysis

## Load

Export analytical datasets:

* fact_sessions.csv
* fact_campaign_daily.csv
* fact_channel_daily.csv

# ROI Analysis

The ROI analysis combines campaign revenue with advertising spend to measure marketing performance.

ROI formula:

ROI = Revenue / Ad Spend

Campaign performance is summarized in:

campaign_roi_summary.csv

This dataset identifies the campaigns generating the highest marketing returns.

---

# Key Insights

Based on the analysis:

### Insight 1 — High ROI campaigns exist

Several campaigns generate strong ROI, indicating efficient marketing investments.

### Insight 2 — Mid-performing campaigns

Some campaigns show moderate ROI (~4–6x). These campaigns could benefit from targeting or creative improvements.

### Insight 3 — Low ROI campaigns

A subset of campaigns shows ROI below 1–2x, indicating inefficient marketing spend.

### Insight 4 — Budget reallocation opportunity

Marketing budget should be shifted from low-ROI campaigns to high-ROI campaigns to maximize revenue growth.
